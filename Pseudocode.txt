Laboratory Activity 1: Add Two Numbers
PSEUDOCODE

START.
INPUT num1.
INPUT num2.
PROCESS sum = num1 + num2
DISPLAY sum.
END.

Laboratory Activity 2: Compute the Average of Three grades
PSEUDOCODE

END IF
END

Laboratory Activity 3: Determine if a Number is Even or Odd
PSEUDOCODE

START
INPUT number.
IF number MOD 2 = 0 THEN
   DISPLAY "Even"
ELSE
   DISPLAY "Odd"
END IF
END

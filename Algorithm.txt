ALGORITHM

Laboratory Activity 3: Determine if a Number is Even or Odd


STEP 1: START
STEP 2: Ask the user to enter a number. Store it as number.
STEP 3: Check if number  MOD 2 equal 0.
STEP 4: If yes, DISPLAY "Even"
STEP 5: If no, DISPLAY "Odd".
STEP 6: End.
